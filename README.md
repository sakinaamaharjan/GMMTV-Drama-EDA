# Exploratory Data Analysis of GMMTV Dramas (2021–2025)
This repository contains a student-led exploratory data analysis project on GMMTV dramas.
The analysis focuses on genre trends, artist popularity, audience ratings, and yearly releases.

## Files
- Exploratory Data Analysis of GMMTV Dramas.pdf – Full analysis report
- GMMTV_Dramas(2021-2025).csv – Cleaned dataset

## Tools Used
- Google Sheets
- Pivot Tables
